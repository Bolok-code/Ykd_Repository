# 阿里云百炼图片能力接入说明

## 已接入的能力

项目现在包含两条相互独立的图片链路：

1. 图片识别与问题解答
   - 微信用户发送图片。
   - `ILinkClient.downloadImageFromMessageItem` 下载并解密图片。
   - `SpringAiBailianVisionClient` 通过 Spring AI 调用百炼 OpenAI 兼容接口。
   - 默认模型为 `qwen3.7-plus`。
   - 模型返回文字描述、图片问答结果或解题步骤。
   - `ILinkClient.sendText` 将结果回复给微信用户。

2. 自然语言文生图
   - 用户无需使用命令，直接发送“帮我生成一张……”“画一幅……”等自然语言。
   - `ILinkMessageRouter` 识别文生图意图。
   - `BailianImageGenerationClient` 调用 Qwen-Image 同步图像生成接口。
   - 默认模型为加速版 `qwen-image-2.0`，默认输出分辨率为 `1024*1024`。
   - 程序立即下载百炼返回的临时图片地址。
   - `ILinkClient.sendImage` 将图片发送给微信用户。

普通文字聊天仍然交给 DeepSeek，天气问题仍然交给天气服务。

## 为什么使用两种调用方式

- 图片识别接口兼容 OpenAI Chat Completions，因此通过 Spring AI 的
  `OpenAiChatModel` 调用。
- Qwen-Image 文生图使用百炼专用的
  `/api/v1/services/aigc/multimodal-generation/generation` 接口，
  并不是 OpenAI `/images/generations` 接口，因此使用独立的百炼适配器调用。
- 两种能力都隐藏在项目接口后面，上层微信路由不依赖具体供应商实现。

官方参考：

- [千问 OpenAI Chat Completions 兼容接口](https://help.aliyun.com/zh/model-studio/qwen-api-via-openai-chat-completions)
- [图像与视频理解](https://help.aliyun.com/zh/model-studio/vision)
- [千问 Qwen-Image 文生图 API](https://help.aliyun.com/zh/model-studio/qwen-image-api)

## 需要填写的本地配置

打开 `config/application-local.yml`，保留已有 DeepSeek 配置并填写：

```yaml
app:
  bailian:
    api-key: sk-你的百炼API-Key
    workspace-id: 你的业务空间ID
  deepseek:
    api-key: 你的DeepSeek密钥
```

`application-local.yml` 已被 `.gitignore` 排除，不会提交到 Git，并且
`application.yml` 会自动加载它。

也可以不填写本地文件，改用环境变量：

```text
DASHSCOPE_API_KEY
BAILIAN_WORKSPACE_ID
```

可选配置：

| 配置项 | 默认值 | 说明 |
|---|---|---|
| `BAILIAN_REGION` | `cn-beijing` | 必须与 API Key、业务空间所在地域一致 |
| `BAILIAN_VISION_MODEL` | `qwen3.7-plus` | 图片识别模型 |
| `BAILIAN_IMAGE_MODEL` | `qwen-image-2.0` | 加速版文生图模型；追求质量可改成 `qwen-image-2.0-pro` |
| `BAILIAN_IMAGE_SIZE` | `1024*1024` | 使用“宽*高”格式；Qwen-Image 2.0 总像素需在 `512*512` 到 `2048*2048` 之间 |
| `BAILIAN_TIMEOUT_SECONDS` | `180` | 识图和生图请求超时 |

## 还需要在百炼控制台确认

1. API Key 属于华北 2（北京）地域。
2. 找到该 API Key 所在的业务空间 ID；它不是阿里云账号 UID。
3. 当前业务空间能够调用 `qwen3.7-plus`。
4. 当前业务空间能够调用 `qwen-image-2.0`；追求更高质量时可改用 `qwen-image-2.0-pro`。
5. 免费额度、账户余额和调用限流满足实际使用需求。
6. 运行机器可以访问百炼 MaaS 地址和百炼返回的阿里云 OSS 临时图片地址。

## 手机上如何验证

使用 JDK 21 构建并启动：

```text
mvn clean package
run.bat chat
```

随后在微信中测试：

1. 发送一张包含题目的图片，应收到文字分析和解题过程。
2. 发送“帮我生成一张雨夜霓虹灯下的橘猫”，应收到一张图片。
3. 发送普通问题，应继续由 DeepSeek 回复。

当前实现每次只处理微信消息中的第一张图片，每次文生图只生成并发送一张图片。
这样可以控制费用、响应时长，并避免一条请求出现两次回复。
