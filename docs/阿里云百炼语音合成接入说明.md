# 阿里云百炼语音合成接入说明

## 当前实现范围

当前实现机器人生成并发送 MP3/WAV 语音文件，不再发送原生微信语音气泡：

```text
微信文字消息
→ ILinkMessageRouter 识别“用语音回答”意图
→ 天气服务或 DeepSeek 生成文字答案
→ BailianSpeechSynthesisClient 调用百炼非实时 TTS
→ 下载 MP3 或 WAV
→ ILinkClient.sendFile 发送音频文件
```

项目已移除 Node.js、silk-wasm、PCM 转换和 `sendVoice()` 链路。语音合成或
文件发送抛出异常时，自动降级为普通文字回复。

注意：iLink 没有微信设备侧投递回执。日志显示“语音文件已提交给 iLink”
只表示上传和接口调用完成，最终是否到账仍需在手机微信中确认。

## 配置

默认配置位于 `src/main/resources/application.yml`：

```yaml
app:
  bailian:
    tts:
      model: ${BAILIAN_TTS_MODEL:qwen-audio-3.0-tts-flash}
      voice: ${BAILIAN_TTS_VOICE:longanhuan_v3.6}
      format: ${BAILIAN_TTS_FORMAT:mp3}
      sample-rate: ${BAILIAN_TTS_SAMPLE_RATE:24000}
      timeout-seconds: ${BAILIAN_TTS_TIMEOUT_SECONDS:180}
```

`BAILIAN_TTS_FORMAT` 仅支持 `mp3` 或 `wav`，默认是体积更小的 `mp3`。
需要 WAV 时可以设置环境变量：

```powershell
$env:BAILIAN_TTS_FORMAT="wav"
```

语音合成复用现有的 `DASHSCOPE_API_KEY`、`BAILIAN_WORKSPACE_ID` 和
`BAILIAN_REGION`。Qwen-Audio-TTS 非实时接口需要北京地域。

## 测试方式

启动微信机器人后发送：

```text
请用语音回答：你好
```

也可以测试天气能力：

```text
请用语音回答：明天杭州天气
```

失败降级顺序：

```text
MP3/WAV 文件 → 普通文字
```

## 错误诊断与重试

语音合成会解析百炼返回的 HTTP 状态、错误码、错误消息和 RequestId。
对网络异常、HTTP 429（非配额不足）和 HTTP 5xx 最多尝试三次，并采用
指数退避；API Key、权限、参数、余额或分配配额错误不会盲目重试。

百炼返回的 OSS 临时下载地址必须原样保留签名查询参数。项目仅把 `http`
协议替换为 `https`，不会重新编码查询参数，避免 `%2B`、`%2F` 等字符被
改写后触发 `SignatureDoesNotMatch`。
