# project-001-CLI-Toolbox

**创建日期：** 2026-07-16
**项目目标：** 构建一个多模块 CLI 命令行工具箱，包含基础命令、天气查询、微信 iLink Bot 与 DeepSeek 智能聊天
**技术栈：** Java 21, Spring Boot 3.5.15, Spring AI 1.1.8, Maven, JUnit 5, OpenILink SDK, DeepSeek Chat Completions API

## 关键决策记录

| 日期 | 决策内容 | 理由 |
|------|----------|------|
| 2026-07-16 | 从 Python 切换到 Java + Maven | 用户已有 JDK21 和 Maven 环境 |
| 2026-07-17 | iLink SDK 切换为 `io.github.lith0924:wechat-ilink-sdk:2.3.3` | 使用正式 Maven 坐标，并获得会话恢复、上下文和生命周期管理能力 |
| 2026-07-17 | 项目改造为非 Web 的 Spring Boot CLI | 使用 Boot 管理启动、依赖注入、配置、日志和关闭生命周期，同时保留原命令行使用方式 |
| 2026-07-17 | DeepSeek 通过 Spring AI `OpenAiChatModel` 接入 | DeepSeek 兼容 OpenAI Chat Completions，并通过 `AiChatClient` 隔离具体模型供应商 |
| 2026-07-17 | 微信聊天按用户串行、跨用户并行 | 保证同一用户的上下文和回复顺序，同时避免模型请求阻塞消息拉取 |
| 2026-07-17 | iLink 层拆分为 `ILinkService` 接口与 `ILinkServiceImpl` 实现 | 让命令层依赖稳定接口，并将具体 OpenILink SDK 实现收敛到 `impl` 包 |
